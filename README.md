# Weather API

This is a simple Express server that demonstrates the use of API and open data.

## .env format

The following code below is an example of this project's `.env` file.

```env
HOSTNAME="127.0.0.1"
HOST="http://127.0.0.1/"
PORT="3000"
NODE_ENV="development"
OPEN_WEATHER_API_KEY="{key here}"
DEBUG="weather-api:*"
```
